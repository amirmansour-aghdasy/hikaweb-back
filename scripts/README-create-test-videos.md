# راهنمای ایجاد ویدئوهای تستی

این اسکریپت سه ویدئو تستی با همان فایل ویدئو اما کاورهای متفاوت ایجاد می‌کند.

## پیش‌نیازها

1. سرور backend باید در حال اجرا باشد
2. حداقل یک فایل ویدئو در مدیا وجود داشته باشد
3. حداقل سه تصویر در مدیا وجود داشته باشد

## نحوه استفاده

### روش 1: استفاده از اسکریپت

```bash
cd back
node scripts/create-test-videos.js
```

### روش 2: استفاده از Dashboard

1. وارد dashboard شوید
2. به بخش Videos بروید
3. روی "افزودن ویدئو جدید" کلیک کنید
4. فایل ویدئو را انتخاب کنید (همان ویدئو موجود)
5. تصویر کاور را انتخاب کنید (یکی از تصاویر موجود)
6. اطلاعات زیر را وارد کنید:

**ویدئو تستی شماره ۱:**
- عنوان فارسی: ویدئو تستی شماره ۱ - آموزش طراحی وب
- عنوان انگلیسی: Test Video 1 - Web Design Tutorial
- توضیحات کوتاه: این یک ویدئو تستی است که برای بررسی عملکرد سیستم ویدئو ایجاد شده است. در این ویدئو به آموزش طراحی وب می‌پردازیم.
- توضیحات کامل: این ویدئو تستی شامل آموزش‌های جامع در زمینه طراحی وب است...
- تگ‌ها: آموزش، طراحی وب، HTML، CSS، JavaScript، تست
- وضعیت: منتشر شده ✓
- ویژه: ✓

**ویدئو تستی شماره ۲:**

- عنوان انگلیسی: Test Video 2 - React Programming

## اطلاعات ویدئوها

هر سه ویدئو از همان فایل ویدئو استفاده می‌کنند اما:
- کاورهای متفاوت دارند (از تصاویر موجود در مدیا)
- عنوان‌های متفاوت دارند
- توضیحات متفاوت دارند
- تگ‌های متفاوت دارند

## بررسی در Frontend

پس از ایجاد ویدئوها، می‌توانید آن‌ها را در frontend بررسی کنید:
- صفحه لیست ویدئوها
- صفحه جزئیات هر ویدئو
- عملکرد پخش ویدئو
- نمایش کاورها
